# Stack & Queue Lab Programs (29-Sep-2025)

This repository contains 9 menu-driven C programs implementing stack, queue, circular queue, and deque structures using arrays and linked lists.

## Programs
1. Integer Stack using Array  
2. Character Stack using Linked List  
3. Integer Linear Queue using Linked List  
4. Linear Queue of Names using Array  
5. Integer Circular Queue using Linked List  
6. Character Circular Queue using Array  
7. Input Restricted Deque using Array  
8. Output Restricted Deque using Array  
9. Odd & Even Stack from Integer Stack using Array  

Each file can be compiled with:
```bash
gcc filename.c -o outputname
./outputname
